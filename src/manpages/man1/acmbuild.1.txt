NAME
acmbuild - acmbuild script
SYNOPSIS
acmbuild [OPTION]
DESCRIPTION
A script to automate AutoConf and Make builds of distribution tarballs.
OPTIONS
-h  prints usage information

-V  prints version information
EXIT STATUS
0
if OK,

1
if minor problems,

64
error in command line arguments.
